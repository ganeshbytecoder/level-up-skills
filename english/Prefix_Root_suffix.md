# Part 1: How Words Are Built

Most advanced English words follow this pattern:

> **Prefix + Root + Suffix**

Example:

**unbelievable**

* un = not
* believe = root
* able = capable of

Meaning:

> not capable of being believed

# Learning Strategy

For vocabulary growth:

1. Learn **prefixes** first (30–40).
2. Learn **50 Latin roots** above.
3. Learn **20 common suffixes**.
4. When you encounter a new word, split it:

   * Prefix
   * Root
   * Suffix

Example:

**Inconceivable**

* in = not
* ceiv (cept) = take/receive
* able = capable

→ "not capable of being accepted as true."

This approach is how many students prepare for GRE, GMAT, SAT, UPSC, CAT, and advanced professional English. With these roots alone, you can infer the meanings of several thousand advanced English words.
